# Vaultocrypt

Vaultocrypt is an internal-first credential and secure note workspace for MetaBox Technology.

## Current State

- `Next.js` App Router foundation
- `shadcn` Luma-based UI setup
- Responsive workspace shell
- Client directory first-pass UI
- Activity, search, and settings placeholder sections

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Next Implementation Areas

- Clerk authentication
- Role-aware access rules
- Database schema and persistence
- Secret encryption helpers
